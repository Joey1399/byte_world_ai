# byte_world_ai

CLI RPG inspired by the supplied HLD/LLD.

## Run

```bash
python main.py
```

## Core Commands

- `help`
- `status`
- `look`
- `sense`
- `move <direction>` (`n/s/e/w/u/d` aliases)
- `inventory`
- `equip <item>`
- `use <item>`
- `read <item>`
- `train <attack|defense|health> [points]`
- `talk <npc>`
- `fight`
- `defend`
- `skill <focus strike|guard stance|second wind>`
- `run`
- `joke`
- `bribe`
- `quest`
- `quit`

## Progression

1. Meet the Wise Old Man.
2. Defeat Giant Frog in the Swamp.
3. Defeat Dragon at the Peak.
4. Optional Ogre cave encounter.
5. Handle the Army of Goblins.
6. Defeat King Makor.
7. Defeat the Onyx Witch.
8. Free and cleanse Elle.
